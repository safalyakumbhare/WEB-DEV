HYPER TEXT MARKUP LANGUAGE

Definition

HTML is the standard markup language for creating Web pages. HTML defines structure of a Web Page like label , paragraph, heading , list ,etc. HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

Basic difference between HTML , CSS AND JavaScript :

HTML :- Defines the structure of a webpage CSS :- Defines the presentation and the style of a webpage JavaScript :- Defines the functionality of a webpage.
